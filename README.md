# SaaS API Automation & Integration Testing

This repository demonstrates professional API testing standards using Postman. It features automated JavaScript test scripts to ensure backend stability and data integrity—critical for SaaS platforms with complex third-party integrations (e.g., Stripe, Shopify).

 Key Features Demonstrated
- Automated Validation:** Custom JS scripts to verify HTTP Status Codes (200 OK, 201 Created).
- Data Integrity:** Schema and payload validation ensuring the server returns exactly what is expected.
- Performance Benchmarking:** Real-time response time monitoring (Target < 500ms).
- crudifecycle:** Full testing of GET (Read) and POST (Create) methods.

 Project Structure
- `SaaS_API_Testing_Portfolio.postman_collection.json`: The full executable test suite.
- `/screenshots`: Visual evidence of passing test cycles.

 How to Run
1. Import the `.json` collection into your **Postman** workspace.
2. Click **Run Collection**.
3. Observe the automated test results in the "Test Results" tab.
